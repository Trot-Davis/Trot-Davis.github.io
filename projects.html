PCB Business Card

This is the project that I'm currently working on. It's a PCB Business card that functions using NFC technology to share my LinkedIn profile as a unique way to network myself. So far, I've created an antenna that has sufficient inductance to ensure the business card is resonating with the NFC's frequency to allow fluent communication. These dimensions and sizing were optimized with the use of Excel as seen above. I have also fabricated both the schematic and PCB libraries in Altium, and designed the electronic schematic as well as layed out the components on the PCB board. All that's left to do is order the PCB and program the NFC chip!

RISC Machine

Created a simple RISC machine that can use read-write memory and perform arithmetic. It works by getting an instruction that gets decoded by the finite state machine, which then proceeds to the datapath. The desired value is computed using Boolean logic, as well as an ALU combinational logic block. From there, I stored the value in memory which can be used in the future. All of the code was done using Visual Studio Code, and was compiled on Quartus. I tested the program by making a testbench and running it in Modelsim to ensure the expected outputs were being output from the instruction that was sent in. I also loaded the program onto an FPGA board (The DE1-SOC) to demo the RISC machine in real life. This was one of my first experiences with hardware and verilog so it was a very enjoyable project for me!

Reflow Solder Oven

This project was one of the most enjoyable experiences of my second year at university. Working alongside five other electrical engineering students, all equally passionate about the field, made every second exciting. Our design worked by reading the temperature from a thermocouple wire placed inside an oven and comparing it to an external voltage reference. This temperature value was then displayed on an LCD screen and transmitted through the COM port to be plotted using a Python program. We set desired times and temperatures for the soak and reflow phases, and once those values were reached, one of the Nuvoton chips signaled both the LCD and the other chip to change states. Various features responded accordingly, and after a lot of trial and error, we successfully implemented the reflow solder oven. My primary role focused on writing the assembly code for the two Nuvoton chips. I designed a finite state machine that controlled the signals sent between the chips and determined which states were displayed on the LCD. I also implemented the timer-based pulse width modulation (PWM) and buzzer control, while the solid heat-up phase was much simpler to program. The second chip received signals from the first and managed the oven's output—switching between heat-up, PWM, and cool-down phases—along with controlling the speaker and an LED. In addition to coding, I helped develop the Python program and worked on wiring the breadboard to ensure everything was connected and functioning properly.

Coin‑Picking Robot

I built a small autonomous robot designed to detect and collect coins from the floor. I integrated sensor inputs for object detection and alignment, designed the motor control and power distribution, and wrote the embedded control logic to navigate, approach, and actuate a pickup mechanism. The project emphasized robust sensing under variable lighting, repeatable motion control, and mechanical‑electrical integration.
