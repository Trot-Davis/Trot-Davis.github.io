Project 1: PCB Business Card - Personal Project

This is the project that I'm currently working on. It's a PCB Business card that functions using NFC technology to share my LinkedIn profile as a unique way to network myself. So far, I've created an antenna that has sufficient inductance to ensure the business card is resonating with the NFC's frequency to allow fluent communication. These dimensions and sizing were optimized with the use of Excel as seen above. I have also fabricated both the schematic and PCB libraries in Altium, and designed the electronic schematic as well as layed out the components on the PCB board. All that's left to do is order the PCB and program the NFC chip!

Project 2: RISC Machine - School Project

Created a simple RISC machine that can use read-write memory and perform arithmetic. It works by getting an instruction that gets decoded by the finite state machine, which then proceeds to the datapath. The desired value is computed using Boolean logic, as well as an ALU combinational logic block. From there, I stored the value in memory which can be used in the future. All of the code was done using Visual Studio Code, and was compiled on Quartus. I tested the program by making a testbench and running it in Modelsim to ensure the expected outputs were being output from the instruction that was sent in. I also loaded the program onto an FPGA board (The DE1-SOC) to demo the RISC machine in real life. This was one of my first experiences with hardware and verilog so it was a very enjoyable project for me!
