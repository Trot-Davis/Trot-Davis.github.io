PCB Business Card

Designed and developed an NFC-enabled business card PCB utilizing 13.56 MHz inductive coupling technology. Implemented custom planar antenna design with optimized trace geometry to achieve target inductance values for resonant operation. Conducted electromagnetic simulation and parametric analysis using spreadsheet modeling to validate antenna performance characteristics. Designed complete schematic and PCB layout in Altium Designer, including component library creation and footprint optimization. Integrated NFC controller IC with power management circuitry and implemented firmware for contactless data transmission. Project demonstrates proficiency in RF circuit design, PCB layout optimization, and embedded systems integration.

RISC Machine

Implemented a 32-bit RISC processor architecture in Verilog HDL featuring instruction decode, finite state machine control unit, arithmetic logic unit (ALU), and dual-port memory interface. Developed comprehensive testbench suite using ModelSim for functional verification and timing analysis. Synthesized design targeting Altera DE1-SoC FPGA platform using Quartus Prime development environment. Achieved successful on-board demonstration with real-time instruction execution and memory operations. Project encompassed RTL design methodology, digital signal processing fundamentals, and FPGA implementation workflows. Demonstrated proficiency in hardware description languages, digital design verification, and embedded systems development.

Reflow Solder Oven

Co-developed dual-microcontroller reflow soldering oven control system with precision temperature profiling capabilities. Architected master-slave communication protocol between Nuvoton MCUs for coordinated thermal management. Implemented assembly language firmware for finite state machine control, LCD interface management, and inter-processor signaling protocols. Designed timer-based pulse width modulation (PWM) control system for heating element regulation with closed-loop feedback from thermocouple sensors. Developed Python-based data acquisition and visualization system for real-time temperature monitoring and profile validation. Integrated buzzer control, LED status indicators, and serial communication interfaces. Project encompassed embedded systems design, thermal control algorithms, real-time data processing, and multi-processor system integration methodologies.
